Hi! I'm Thomas Augestad Weholt — an avid Python coder, free software enthusiast, hobby photographer, father of two, and husband of one.

I got my first computer at 12 (Commodore 64), leveled up to an Amiga 500, and bought my first PC in 1993. I started programming in 1994 and have worked in tech since 1999. I've been building software professionally since 2000.

## Professional Experience

I've spent most of my career in backend-heavy enterprise work, with frontend sprinkled in when needed.

Technologies used professionally (self-assessed):

- Borland Delphi (2)
- Microsoft SQL Server (4)
- JavaScript (3)
- C# (4–5)

## Free Software

I got into Linux in 1996. A classmate introduced me to Perl and Python. Perl had a short run in my life; Python became the long-term relationship.

A few years later I discovered Django (pre-1.0 era), and its documentation plus community made me stay. Since then, Django has powered most of my hobby and OSS projects.

## Python — The Love Affair

Python still feels like the shortest distance between idea and working code. The projects that have influenced me most include Django, Wagtail, HTMX, Docker, Requests, Pillow, PyTest, and MkDocs Material.
